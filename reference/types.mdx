# Types

This page is the public type reference for the MDX docs. The source of truth is `common/src/types.rs`.

## Scalar conventions

| Type | Meaning |
| --- | --- |
| `Address` | Soroban contract or account address. |
| `i128` | Token amounts, fixed-point values, caps, rates, and fees. |
| `u32` | Category IDs, enum discriminants, decimals, TWAP count, position limits. |
| `u64` | Account IDs and timestamps. |
| `Symbol` | Soroban symbol, used for oracle tickers and event action tags. |
| `Vec<T>` | Soroban vector. |
| `Map<K, V>` | Soroban map. |
| `Option<T>` | Optional value. |

## Enums

| Enum | Variants |
| --- | --- |
| `AccountPositionType` | `Deposit = 1`, `Borrow = 2` |
| `PositionMode` | `Normal = 0`, `Multiply = 1`, `Long = 2`, `Short = 3` |
| `OracleType` | `None = 0`, `Normal = 1` |
| `ExchangeSource` | `SpotOnly = 0`, `SpotVsTwap = 1`, `DualOracle = 3` |
| `ReflectorAssetKind` | `Stellar = 0`, `Other = 1` |
| `MarketStatus` | `PendingOracle = 0`, `Active = 1`, `Disabled = 2` |
| `SwapVenue` | `Soroswap`, `Aquarius`, `Phoenix`, `NativeAmm`, `StaticBridge` |

## Strategy swap types

| Type | Fields |
| --- | --- |
| `SwapHop` | `fee_bps`, `pool`, `token_in`, `token_out`, `venue` |
| `SwapPath` | `amount_in`, `hops`, `min_amount_out` |
| `AggregatorSwap` | `paths`, `total_min_out` |
| `BatchSwap` | `paths`, `sender`, `total_min_out` |

`BatchSwap` is internal to the controller-router call. Public strategy entrypoints accept `AggregatorSwap`; the controller fills `sender`.
