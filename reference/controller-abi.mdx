# Controller ABI

The controller is the public protocol ABI. Pools are implementation contracts owned by the controller.

## Access and lifecycle

| Function | Auth | Returns | Purpose |
| --- | --- | --- | --- |
| `__constructor(admin)` | one-shot | `()` | Sets owner/admin, grants `KEEPER`, initializes position limits, and pauses the controller. |
| `upgrade(new_wasm_hash)` | Owner | `()` | Pauses and upgrades controller WASM. |
| `pause()` | Owner | `()` | Pauses user and role-gated flows that carry `when_not_paused`. |
| `unpause()` | Owner | `()` | Unpauses after deployment/configuration. |
| `grant_role(account, role)` | Owner | `()` | Grants role through `stellar-access`. |
| `revoke_role(account, role)` | Owner | `()` | Revokes role. |
| `has_role(account, role)` | View | `bool` | Checks role membership. |
| `transfer_ownership(new_owner, live_until_ledger)` | Owner | `()` | Starts two-step owner/admin transfer. |
| `accept_ownership()` | Pending owner | `()` | Accepts ownership and syncs admin roles. |

## User operations

| Function | Auth | Returns | Notes |
| --- | --- | --- | --- |
| `supply(caller, account_id, e_mode_category, assets)` | `caller.require_auth()` | `u64` | Creates an account when `account_id == 0`; returns resolved ID. |
| `borrow(caller, account_id, borrows)` | Account owner | `()` | Strict oracle mode; validates LTV and HF. |
| `withdraw(caller, account_id, withdrawals)` | Account owner | `()` | Amount `0` means withdraw all. |
| `repay(caller, account_id, payments)` | Payer auth | `()` | Permissionless on target account; overpayment refunded to payer. |
| `liquidate(liquidator, account_id, debt_payments)` | Liquidator auth | `()` | Requires `HF < 1e18`. |
| `flash_loan(caller, asset, amount, receiver, data)` | Caller auth | `()` | Receiver callback must repay `amount + fee`. |

## Strategy operations

| Function | Auth | Returns | Purpose |
| --- | --- | --- | --- |
| `multiply(caller, account_id, e_mode_category, collateral_token, debt_to_flash_loan, debt_token, mode, swap, initial_payment, convert_swap)` | Caller auth | `u64` | Opens or extends leveraged position. |
| `swap_debt(caller, account_id, existing_debt_token, amount, new_debt_token, swap)` | Account owner | `()` | Replaces one debt asset with another. |
| `swap_collateral(caller, account_id, current_collateral, amount, new_collateral, swap)` | Account owner | `()` | Replaces one collateral asset with another. |
| `repay_debt_with_collateral(caller, account_id, collateral_token, collateral_amount, debt_token, swap, close_position)` | Account owner | `()` | Uses collateral to repay debt; optional close. |

## Keeper and revenue

| Function | Role | Returns | Purpose |
| --- | --- | --- | --- |
| `update_indexes(caller, assets)` | `KEEPER` | `()` | Syncs market indexes for selected assets. |
| `update_account_threshold(caller, asset, has_risks, account_ids)` | `KEEPER` | `()` | Updates affected account thresholds after risk changes. |
| `clean_bad_debt(caller, account_id)` | `KEEPER` | `()` | Cleans qualifying bad debt. |
| `keepalive_shared_state(caller, assets)` | `KEEPER` | `()` | Extends shared controller state. |
| `keepalive_accounts(caller, account_ids)` | `KEEPER` | `()` | Extends account metadata and side maps. |
| `keepalive_pools(caller, assets)` | `KEEPER` | `()` | Delegates TTL extension to pools. |
| `claim_revenue(caller, assets)` | `REVENUE` | `Vec<i128>` | Pulls pool revenue and forwards to accumulator. |
| `add_rewards(caller, rewards)` | `REVENUE` | `()` | Adds supplier rewards to pools. |

## Market and pool configuration

| Function | Auth | Returns | Purpose |
| --- | --- | --- | --- |
| `set_aggregator(addr)` | Owner | `()` | Sets strategy router. |
| `set_accumulator(addr)` | Owner | `()` | Sets revenue sink. |
| `set_liquidity_pool_template(hash)` | Owner | `()` | Sets pool WASM template hash. |
| `approve_token_wasm(token)` | Owner | `()` | Allows token address for market creation. |
| `revoke_token_wasm(token)` | Owner | `()` | Removes token approval. |
| `create_liquidity_pool(asset, params, config)` | Owner | `Address` | Deploys one pool for `asset`; market starts `PendingOracle`. |
| `edit_asset_config(asset, cfg)` | Owner | `()` | Updates risk flags and caps except e-mode enablement. |
| `upgrade_pool_params(asset, params)` | Owner | `()` | Alias for `upgrade_liquidity_pool_params`. |
| `upgrade_liquidity_pool_params(asset, params)` | Owner | `()` | Syncs indexes and updates pool interest-rate params. |
| `upgrade_pool(asset, new_wasm_hash)` | Owner | `()` | Alias for `upgrade_liquidity_pool`. |
| `upgrade_liquidity_pool(asset, new_wasm_hash)` | Owner | `()` | Upgrades a pool's WASM. |
| `set_position_limits(limits)` | Owner | `()` | Sets account max supply and borrow positions. |
