# Anti-rationalizations

| Phrase | Why it fails | Do this instead |
|--------|--------------|-----------------|
| "I get the gist" | Gist misses ordering and storage couplings | Line-by-line blocks |
| "I'll remember the invariant" | Context decays across agents | Write `GLOBAL_MODEL.md` |
| "External call is probably fine" | External is adversarial until proven | Jump in or Case B |
| "Skip this helper" | Helpers encode assumptions | Fold into parent agent |
| "Broad sweep first for orientation" | Orientation without artifacts is a skim | Inventory table only |
| "Too many functions" | Unbounded scope collapses to skim | Wave-1 ≤8, then stop |
| "Found a High mid-context" | Premature findings skip refutation | `OPEN_QUESTIONS.md` |
| "One agent for the whole crate" | Recreates the failure mode | One agent per queued fn |
